Initial public release. Includes the Vue primitives (`useCurrentFrame`, `useVideoConfig`, `defineComposition`, `Sequence`, `Audio`, `delayRender`/`continueRender`), the scrubber preview, a Puppeteer + FFmpeg render pipeline with parallel worker pool, deterministic per-frame re-seeding, and the `vumo render` CLI.
